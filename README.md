The dataset used in this project is the Diabetes Prediction Dataset from Kaggle:
https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset
This dataset contains 100,000 records with 9 features related to health and lifestyle factors.
The 9 columns represents attributes like 
1.gender
2.age
3.bmi 
4.smoking history
5.HbA1c level 
6.hypertension 
7.heart disease 
8.blood glucose level
9.diabetes
This dataset is used to analyze patterns and identify key factors influencing diabetes through data visualization.
